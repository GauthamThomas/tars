Add your audio files here as .wav or .mp3 files.

Reference them in triggers.yaml by filename.
Example: audio: tars_necessary.wav

If a file is missing, TARS plays a beep instead so triggers still fire.

Recommended sources for audio clips:
- Record your own using any voice recorder
- Extract from video files using ffmpeg
- Use a text-to-speech service for custom responses

Note: For personal use only. Do not distribute copyrighted audio.
